# TanStack Start Template

* TanStack Start
* Tailwind CSS v4
* shadcn/ui
* ElysiaJS
* Drizzle
* Better Auth
* ky

## Quick Start

```bash
# clone this repo
git clone https://github.com/myparsleycat/tanstack-start-template

# install
pnpm install

# database Setup
pnpm dlx drizzle-kit push

# development
pnpm dev

```
